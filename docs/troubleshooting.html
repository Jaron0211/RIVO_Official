Troubleshooting

Common Issues

"Invalid or expired token"

Cause: Access Token has expired (default: 30 minutes).

Solution: Use the Refresh Token to obtain a new Access Token via /api/v1/auth/refresh.

"Connection refused"

Cause: Server not running, wrong port, or firewall blocking.

Solution: Check if the process is running and verify server.port in config.yaml.

"Database connection failed"

Cause: Database unreachable or invalid credentials.

Solution: Verify the database DSN and ensure the database service or file is accessible.

"Rate limit exceeded"

Cause: Too many requests from a single IP.

Solution: Implement exponential backoff retry on the client, or adjust requests_per_second in config.

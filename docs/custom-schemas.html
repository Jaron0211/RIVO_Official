Custom Message Schemas

RIVO allows you to extend the API interface via JSON Schema. This enables robots to send custom-format data without modifying server code.

7.1 Schema Directory Structure

All schema files are stored in the custom_msg/ directory. The manifest file determines which schemas are loaded.

custom_msg/
├── _manifest.txt          # Active schema list
├── control_input.json     
└── sensor_reading.json    # Your custom schema

7.2 Manifest File

_manifest.txt example:

# Built-in
control_input.json

# Custom
sensor_reading.json

7.3 Adding a Schema

  1. Create a JSON file (following JSON Schema Draft-07 format).
  2. Add the filename to _manifest.txt.
  3. Restart RIVO Server.

The new API endpoint will automatically map to /api/v1/robots/{id}/<schema_id>.

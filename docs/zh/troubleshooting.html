疑難排解

常見問題

"Invalid or expired token"

原因： Access Token 已逾期（預設為 30 分鐘）。

解決方案： 使用 Refresh Token 透過 /api/v1/auth/refresh 終端取得新的 Access Token。

"Connection refused"

原因： 伺服器未運行、埠號錯誤或防火牆阻擋。

解決方案： 檢查進程是否正在執行，並核對 config.yaml 中的 server.port 設定。

"Database connection failed"

原因： 資料庫無法訪問或憑據錯誤。

解決方案： 驗證資料庫 DSN 並確保資料庫服務或檔案可被存取。

"Rate limit exceeded"

原因： 單一 IP 發送了過多請求。

解決方案： 在客戶端實作指數退避（Exponential Backoff）重試機制，或調整設定檔中的 requests_per_second。
